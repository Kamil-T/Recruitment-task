## Prerequisites

```sh
Before you begin, ensure you have met the following requirements:
- You have installed Node.js, newest versions include npm
- You have installed yarn or npm
```

## Install

```sh
npm install
or
yarn install
```

## Usage

```sh
npm start
or
yarn start
```

## Run tests

```sh
npm test
or
yarn test
```
